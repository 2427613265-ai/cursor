# 权利要求书

## 权利要求 1

一种无人机马达散热外壳，其特征在于，包括中心安装盘、加强筋、环形外沿和导流齿；

所述中心安装盘为圆盘状，中心设有轴孔，所述轴孔周围设有安装孔；

所述中心安装盘的外侧连接有所述加强筋，相邻加强筋之间形成镂空通风槽，所述镂空通风槽的侧缘设有向内翻折的引风唇边；

所述加强筋的外侧连接有所述环形外沿，所述环形外沿的外侧面设有所述导流齿，所述导流齿沿所述环形外沿的轴向延伸。

---

## 独权撰写说明（内部参考，不入申请文件）

### 解决的技术问题

传统马达散热外壳散热效率低，气流难以有效进入电机内部形成稳定对流通道。

### 必要技术特征取舍

| 特征 | 是否写入独权 | 理由 |
|---|---|---|
| 中心安装盘 + 轴孔 + 安装孔 | 是 | 外壳与转子装配所必需 |
| 加强筋 + 镂空通风槽 | 是 | 轻量化与通风通道所必需 |
| 引风唇边 | 是 | 与导流齿形成泵吸协同的核心区别特征 |
| 环形外沿 + 导流齿 | 是 | 旋转导流所必需 |
| 非对称截面导流齿 | 否→从权 | 对称齿亦可实现基本导流，非解决技术问题所必需 |
| 内侧凹形引风曲面 | 否→从权 | 引风唇边已建立引风结构，凹形曲面为优选实施方式 |
| 轴向错层迷宫通风槽 | 否→从权 | 防尘防溅的优选方案，非基本散热所必需 |
| 加强筋径向渐宽 | 否→从权 | 导热优化细节，非必要 |
| 数量（8个）、均匀分布 | 否→从权 | 具体实施方式，缩小保护范围 |

### 待布局从权方向

1. 镂空通风槽为轴向错层迷宫式（上下孔周向错开、非直通）
2. 导流齿截面为非对称结构（迎风侧陡立面 + 背风侧缓坡面）
3. 加强筋内侧设有凹形引风曲面，与引风唇边衔接
4. 加强筋沿径向由内向外宽度渐增
5. 加强筋截面为流线型
6. 导流齿沿周向呈螺旋状布置或高度渐变
7. 中心安装盘端面设有环形定位凸台
8. 材料、一体成型、表面处理等
