## 二、交底书初版独立权利要求（忠实转化，尚未按主要技术问题定向修订）

> 说明：以下仅把交底书已有内容写成规范独权表述，**未引入补强新特征**。

**1.** 一种具有双侧双单向阀与独立过冷支路的可变流路换热器，其特征在于，包括输入管组件、输出管组件、过渡管组件、换热器本体、双侧双单向阀和单向阀；  
所述换热器本体内设有 N 条主换热支路和 1 条专用过冷支路，总计 N+1 条换热管路，其中 N 为大于或等于 2 的正整数；所述 N 条主换热支路并联布置，所述换热器本体内不设置分隔隔板；  
所述输入管组件与所述换热器本体的一端连通，并与所述双侧双单向阀连接；所述双侧双单向阀还与所述过渡管组件连接；  
所述输出管组件与所述换热器本体的另一端连通，所述输出管组件配置有所述单向阀，所述单向阀与所述过渡管组件连接；  
所述 N 条主换热支路的末端经分配器后与所述专用过冷支路及所述单向阀连通；所述专用过冷支路单独设置并承担制冷工况下的过冷换热，且所述专用过冷支路的管路长度可独立于所述主换热支路进行设计；  
所述双侧双单向阀具有上端阀芯和下端阀芯，所述上端阀芯的导通方向与所述单向阀的导通方向一致；  
在制冷工况下，系统高压使所述上端阀芯和所述单向阀关闭，冷媒并联流经所述 N 条主换热支路冷凝后，经所述输出管组件汇入所述专用过冷支路过冷，再经所述下端阀芯流入所述过渡管组件；  
在制热工况下，系统低压使所述单向阀和所述上端阀芯导通、并使所述下端阀芯关闭，冷媒经所述 N 条主换热支路与所述专用过冷支路全部并联蒸发后，由所述输入管组件汇出。

## 三、技术问题梳理与主要技术问题（待确认）

### 1. 交底书方案可对应解决的技术问题

| 层级 | 技术问题 | 主要对应特征 |
|------|----------|--------------|
| A（核心） | 可变流路换热器制冷作冷凝器时过冷段偏长，液态管路压降高、过冷度偏高，制冷能效下降 | 唯一专用过冷支路；过冷管长独立匹配；制冷时 N 并联冷凝 + 独立短流程过冷 |
| B | 制热作蒸发器时流通面积不足、两相压降大、分流不均/局部干烧 | 制热时 N+1 全域并联；无源阀切换 |
| C | 传统可变流路依赖电磁阀/电控，结构复杂、适配性差 | 双侧双单向阀 + 单向阀压差无源切换 |
| D（交底书批评的两类现有结构） | “两路过冷”“分配器内换向”带来长流程过冷 | 不采用两路共用过冷、不依赖分配器内换向 |

### 2. 拟定的主要技术问题（请确认或调整）

**主要技术问题：**  
在冷暖两用可变流路换热器中，如何在制热工况仍能实现多支路大流通面积并联蒸发的同时，使制冷工况获得长度可独立匹配的短流程专用过冷段，从而避免过冷段偏长导致的液态压降偏高与过冷度偏高。

**选定理由：**  
- 与交底书自称的核心痛点一致；  
- 相对对比文件1（侧重“制冷/制热最佳流路数不同 + 集管内分液”），本案主张的是**相反的冷凝器流路哲学**：多支路并联冷凝 + **唯一短流程独立过冷**，而非减少支路、拉长串联流程；  
- 便于把“专用过冷支路 + 阀组切换拓扑”写成独权必要特征，同时把均液细节、阀体细部放到从权。

## 四、相对对比文件1（CN118548543A）的创造性评估（倾向性）

### 1. 对比文件1在教什么

- 第一/第二集管组件内置**不对称单向阀**，靠流向实现可变流路；  
- **作蒸发器**：多支路并联，降两相压降；  
- **作冷凝器**：减少并联支路、形成更长流程，提高管内流速与换热系数；  
- 重点还包括集管内加速均液、不对称阀芯等。

### 2. 与交底书的相同点（风险区）

- 均用单向阀 + 冷媒流向做**无源可变流路**；  
- 均在换热器两侧集管/管组件布置阀；  
- 均实现“一种工况多支路并联、另一工况流路重组”。

### 3. 潜在区别技术特征（机会点）

| 区别点 | 交底书 | 对比文件1 | 创造性意义 |
|--------|--------|-----------|------------|
| D1 制冷过冷组织方式 | N 并联冷凝 + **1 条专用短过冷支路** | 冷凝器侧**减支路、加长串联流程** | 问题导向不同；若拓扑与效果能证明，可构成非显而易见改进 |
| D2 过冷段独立性 | 唯一专用过冷支路，管长独立设计 | 未提出“专用过冷支路/独立匹配过冷长度” | 需用结构特征写清，不能只写效果 |
| D3 阀组形态 | 一侧“双侧双单向阀”（上下反向），另一侧单向阀 + 分配器外置 | 两侧集管内各一只不对称单向阀 | 若仅为普通单向阀外置组合，易被视作常规；需补结构特殊性或不可替换的拓扑必要性 |
| D4 芯体分隔 | 强调无内隔板 | 集管腔室分隔 + 可变连通 | 单独作为区别较弱，宜作辅助特征 |

### 4. 结论（带不确定性标注）

- **实用新型**：在补齐连接关系与可实施细节后，**较有希望**以“独立过冷支路 + 特定阀组切换拓扑”获得授权。  
- **发明**：仅凭现有交底书文字，审查员很可能会说：在对比文件1可变流路基础上，把某一支路改作冷凝器末段过冷、并用单向阀切换并联/串联，属于**常规流路改型**；**发明授权风险偏高**，必须补强（更深问题 + 更不可替换的结构/控制逻辑 + 对比实验）。  
- 本评估仅基于用户提供的对比文件1及交底书，**未做全库检索**；若还存在更近的“独立过冷支路可变流路”现有技术，结论可能更严。
