# 【第5步】2.3 显著的技术进步 + 第二节收束

2.3、修改后的权利要求1具有显著的进步

修改后的权利要求1在技术效果上与前述实际解决的技术问题相对应，具体体现在：先在第k时刻根据室内实时灯光照度，经灯光亮度调控模型将实际灯光亮度调整至初步期望值，使照度条件先落入可用区间；再在第k+1时刻依据空气干球温度与相对湿度按特定公式算出温差值E，并经照度色温耦合调控模型，按E所处区段同时给出色温调控量与耦合亮度调节量，将实时灯光色温与实际灯光亮度一并调向目标期望值；其后等待预设时长并重复上述过程，使室内目标空间维持在“灯光智控”模式。整条闭环把“照度—亮度初步调节”与“体感温差—色温及耦合亮度同步调节”前后衔接起来，而不是只完成其中一路控制。

由此，本申请能够在满足室内照度需求的同时，按体感冷热偏向调整色温，并在色温变化时按照度条件对亮度作与之匹配的耦合修正，避免只调色温、不顾照度与色温搭配，或只按照度调亮度、不顾人体热感觉的情形。说明书亦指出，在空调季节还可通过色温相关的热感觉调节，减少用户过度依赖空调冷热设定的倾向，从而在光环境舒适之外带来一定的节能协助效果。这些效果依附于修改后权利要求1所限定的分时两级结构、温差E的具体算法以及照度色温耦合调控模型，并非对比文件1仅作照度—亮度闭环、或对比文件2仅作温湿度—色温调节所能直接体现。

综上所述，修改后的权利要求1具有突出的实质性特点和显著的进步，具备创造性，符合专利法第二十二条第三款的规定。
