# 一种具有可移动钻探设备的勘测平台

> 权利要求书优化稿（基于用户初稿《实用新型撰写模版.doc》）  
> 优化范围：仅权利要求书；说明书其他部分未同步改写。

## 一、质检要点与修改说明

### 1. 形式合规 / 依附编号（硬伤，已改）

| 原权利要求 | 问题 | 处理 |
|---|---|---|
| 原权5（刻度标识）引用权3 | 使用了仅在权2首次出现的“精调导轨”，依附链条不含该名词 | 改为引用权2，并在特征部分引入滑移座 |
| 原权6（锁紧装置）引用权3 | 同上，锁紧对象是精调导轨，不依赖滑移座 | 改为引用权2 |

### 2. 引用结构

- 导轨细分（权2）、滑移座（权3）、三月池（权7）、多台钻探设备（权9）、推进装置（权10）均为相对独权的**独立改进**，保持平行引用权1。
- 条状月池（权4）、刻度（权5）、锁紧（权6）依赖“精调导轨”，**串联引用权2**。
- 三池具体方位（权8）依赖权7，串联引用。

### 3. 用语与清楚性

- “竖向移池导轨”改为“纵向移池导轨”：甲板平面内与横向对应的应为船长/长度方向，避免被理解为铅垂方向。
- 权2按“先列部件、后展开”改写；横向/纵向移池导轨与精调导轨的衔接关系写清为“分别衔接”。
- 独权补充“月池开设于平台本体”，避免月池与平台本体的位置关系仅靠“包括”暗示。
- 独权末句由纯效果状语收束为与月池钻孔位置直接关联的结构能力表述。

### 4. 独权范围观察（未强行改写发明点）

- 现独权以“导轨沿月池延伸 + 钻探设备可沿导轨移动并经月池钻孔”为核心，与说明书技术问题基本对应，必要特征大体齐全。
- 精调/移池分轨、条状月池、锁紧、刻度等具体实现已下放从权，有利于保护范围。
- 推进装置（权10）与“不移船即可多孔位钻孔”关联较弱，属外围配套；保留作从权，不并入独权。
- 若后续检索发现“月池旁滑移钻机”接近，可考虑将“锁紧装置”或“条状月池”择一并入独权以加强创造性；本次按初稿布局优化，不擅自抬高独权门槛。

## 二、优化后的权利要求书

1. 一种具有可移动钻探设备的勘测平台，其特征在于，包括平台本体、至少一个月池、导轨以及钻探设备；所述月池开设于所述平台本体；所述导轨设置于所述平台本体上，且所述导轨的至少一部分沿所述月池延伸；所述钻探设备可移动地设置于所述导轨上，并能够沿所述导轨移动至不同的水平位置经所述月池进行钻孔。

2. 根据权利要求1所述的具有可移动钻探设备的勘测平台，其特征在于，所述导轨包括精调导轨和移池导轨；所述精调导轨沿所述月池设置；所述移池导轨包括横向移池导轨和纵向移池导轨；所述横向移池导轨与所述纵向移池导轨相交，所述横向移池导轨和所述纵向移池导轨分别与所述精调导轨衔接。

3. 根据权利要求1所述的具有可移动钻探设备的勘测平台，其特征在于，所述导轨上可移动地设有滑移座，所述钻探设备安装于所述滑移座上，并由所述滑移座带动沿所述导轨移动。

4. 根据权利要求2所述的具有可移动钻探设备的勘测平台，其特征在于，所述月池沿所述精调导轨的延伸方向呈条状。

5. 根据权利要求2所述的具有可移动钻探设备的勘测平台，其特征在于，所述导轨上可移动地设有滑移座，所述钻探设备安装于所述滑移座上；所述精调导轨上设有刻度标识，所述滑移座上设有与所述刻度标识配合的指示标记。

6. 根据权利要求2所述的具有可移动钻探设备的勘测平台，其特征在于，所述精调导轨上设有锁紧装置，所述锁紧装置用于在钻孔作业时将所述钻探设备相对于所述精调导轨锁定。

7. 根据权利要求1所述的具有可移动钻探设备的勘测平台，其特征在于，所述平台本体上开设有三个月池，所述导轨连通各所述月池；所述钻探设备能够沿所述导轨移动至不同的所述月池所对应的位置。

8. 根据权利要求7所述的具有可移动钻探设备的勘测平台，其特征在于，所述三个月池分别为第一月池、第二月池和第三月池；所述第一月池设置于所述平台本体宽度方向一端的中部；所述第二月池和所述第三月池分别设置于所述平台本体长度方向两侧的中部。

9. 根据权利要求1所述的具有可移动钻探设备的勘测平台，其特征在于，所述导轨上可移动地设置有至少两台所述钻探设备。

10. 根据权利要求1所述的具有可移动钻探设备的勘测平台，其特征在于，所述平台本体上还设有推进装置；所述推进装置包括尾部推进器和侧向推进器；所述尾部推进器设置于所述平台本体的尾部，所述侧向推进器设置于所述平台本体的首部。

## 三、引用关系示意

```
权1 独权（平台本体 + 月池 + 导轨 + 可移钻探设备）
 ├─ 权2 精调导轨 + 移池导轨（横/纵相交并分别衔接精调）
 │    ├─ 权4 月池沿精调导轨呈条状
 │    ├─ 权5 滑移座 + 刻度/指示标记
 │    └─ 权6 精调导轨锁紧装置
 ├─ 权3 滑移座（不依赖分轨，保护较宽）
 ├─ 权7 三个月池 + 导轨连通
 │    └─ 权8 三池甲板方位布置
 ├─ 权9 至少两台钻探设备
 └─ 权10 尾部/侧向推进器
```
