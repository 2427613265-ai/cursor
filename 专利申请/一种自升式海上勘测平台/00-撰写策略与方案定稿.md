# 撰写策略与方案定稿

## 一、对比结论（交底书 vs CN215884006U）

原交底书《一种深远海风电勘测试验平台》的核心仍是“三孔月池一次站桩三孔”，与对比文件 CN215884006U 高度重合；75 m / 25 天 / 60 人、信息系统等难以稳定支撑实用新型创造性。

## 二、本案主线（已定）

| 项 | 定稿内容 |
|---|---|
| 专利名称 | 一种自升式海上勘测平台 |
| 专利类型 | 实用新型（形状/构造） |
| 主要技术问题 | 桩腿站定后，固定方孔月池难以相对目标孔位做结构级位置微调，对位精度不足 |
| 核心手段 | 条状月池 + 沿月池延伸方向布置的导轨 + 可移动钻机单元 + 作业位锁定 |

## 三、五维度策略摘要

1. **商业/抵御**：重点覆盖平台制造方及钻探设备布置方案提供方，防止他人在自升式勘测平台上采用“长条月池+滑移钻机”规避三孔固定布局专利。
2. **保护范围**：独权只保留解决对位问题的最少结构特征；具体驱动方式、横向微调、辅孔保留等下放从权。
3. **侵权判定**：强调可见结构（月池开口形状、导轨、滑移座、锁定件），避免把控制算法写成实用新型必要特征。
4. **规避与退路**：从权覆盖纵向/横向条状、双侧导轨、齿条/液压驱动、横向微调轨道、辅月池、刻度定位等；说明书末公开变劣方案（仅离散多工位无连续滑移等）占位。
5. **布局**：单独立权（产品）；交底书中深远海参数、生活模块、信息系统等未入权内容写入说明书末充分公开，不强行塞进独权。

## 四、创造性倾向评估（无完整检索，仅倾向）

相对 CN215884006U：区别特征为条状月池 + 导轨可移钻机 + 锁定，解决的是“站桩后微调对位”，与对比文件“一次站桩三孔少移船”问题不同。  
**倾向**：具备实用新型所需的实质性特点和进步；但海上“滑移钻机/可移井架”在石油平台领域可能存在相近技术，**申报前务必补充检索**（关键词建议：moon pool skid / 月池 滑移钻机 / 自升式 移动钻机 轨道）。
