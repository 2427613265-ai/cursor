# 权利要求质检

本分支用于存放**对已撰写完成的权利要求书进行审查/质检**相关的 skill，与"专利撰写"（落笔写申请文件）、"说明书质检"（审查说明书具体实施方式）并列。

## 已有 skill

| skill | 用途 | 典型触发 |
|-------|------|----------|
| `权利要求书质检` | **整份**权利要求书质检（主控 skill） | "质检权利要求书""审核整份权利要求" |
| `单条权利要求审查` | **单条**权利要求审查 | "检查权2""审阅权利要求3" |
| `权利要求引用结构检查` | 引用结构检查（平行/串联） | 由权利要求书质检必读调用，一般不单独触发 |

## 推荐调用关系

```
整份质检：
  权利要求书质检（主控）
    → 权利要求形式合规性规范（必读）
    → 权利要求引用结构检查（必读）
    → 按需加载撰写规范子 skill

单条审查：
  单条权利要求审查
    → 按需引用权利要求形式合规性规范、权利要求清楚性与逻辑一致性规范等
    → 若影响全文编号/引用，提醒转整份质检
```

新增相关 skill 时，请在本目录下按照 `.cursor/skills/专利/权利要求质检/<中文名称>/SKILL.md` 的结构添加。
