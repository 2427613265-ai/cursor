# 审查意见答复

本分支用于存放**答复审查意见通知书**相关的 skill，与"专利撰写"（撰写申请文件）、"权利要求质检"（对已撰写权利要求书的质检）是并列的顶层分支，对应专利审查阶段的工作。

## 现有 skill

| skill | 路径 | 用途 |
|-------|------|------|
| 创造性答复意见陈述书撰写 | `创造性答复/意见陈述书撰写/` | 针对创造性（法22.3）**分步**撰写。先针对审查意见原文逐条答复（可交织特征/问题/动机/启示，**有益效果不写在答复段**）；独权创造性论述固定为 **2.1 区别技术特征 → 2.2 非显而易见 → 2.3 显著技术进步（有益效果统一写此）**。区别特征须编号+权利要求原文；界定技术问题后对比审查员概括；情形一将补充内容接到原独权末尾。先分析选法（通用手法，非案情特例）并用 plan 驱动逐步撰写；禁止一次出全文 |
