# 审查意见答复

本分支用于存放**答复审查意见通知书**相关的 skill，与"专利撰写"（撰写申请文件）、"权利要求质检"（对已撰写权利要求书的质检）是并列的顶层分支，对应专利审查阶段的工作。

## 现有 skill

| skill | 路径 | 用途 |
|-------|------|------|
| 创造性答复意见陈述书撰写 | `创造性答复/意见陈述书撰写/` | 针对创造性（法22.3）**分步**撰写。**一通**以常规三段论为骨架，审查员观点嵌入 2.2；**二通**以针对审查员核心观点作答为主，三段论收束。**修改策略**：最核心权利要求并入独权并为二通留余地。手法组合成递进链，但**内在合理性优先于套模板**（不成立的失败预期/互斥等不得硬写）。技术问题须**问题要素导向、禁止手段植入**；非显而易见性以该问题为锚否定技术启示。先判一通/二通、选法组合并用 plan 驱动；禁止一次出全文 |
