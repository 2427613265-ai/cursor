# 审查意见答复

本分支用于存放**答复审查意见通知书**相关的 skill，与"专利撰写"（撰写申请文件）、"权利要求质检"（对已撰写权利要求书的质检）是并列的顶层分支，对应专利审查阶段的工作。

## 现有 skill

| skill | 路径 | 用途 |
|-------|------|------|
| 创造性答复意见陈述书撰写 | `创造性答复/意见陈述书撰写/` | 针对创造性（法22.3）**分步**撰写意见陈述书：融合「逐条驳斥审查员结论性观点（含反驳手法工具箱）」与「创造性三段论正面论证」；禁止一次出全文，确认后组装 |

## 可继续细分子分支（尚未建立）

- 新颖性答复
- 修改超范围答复
- 权利要求不清楚/得不到说明书支持的答复
