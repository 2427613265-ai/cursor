# 审查意见答复

本分支用于存放**答复审查意见通知书**相关的 skill，与"专利撰写"（撰写申请文件）、"权利要求质检"（对已撰写权利要求书的质检）是并列的顶层分支，对应专利审查阶段的工作。

## 现有 skill

| skill | 路径 | 用途 |
|-------|------|------|
| 创造性答复意见陈述书撰写 | `创造性答复/意见陈述书撰写/` | 针对创造性（法22.3）**分步**撰写意见陈述书。核心是**深度融合**：在回应每一条审查意见的**同一段论证**里，把区别技术特征/实际技术问题/结合动机/技术启示/非显而易见/有益效果交织成连贯论证链，而非“驳斥一节+三段论另一节”分块拼接。拿到审查意见先判明是否修改权利要求、分析选用哪几种反驳手法（非全用），用 plan 待办清单列出全部步骤/子项并逐步标记进度，按用户提示一步步撰写；区分 A 档核心必达要点与 B 档视情选用手法（含相反技术教导）；禁止一次出全文，确认后组装 |
