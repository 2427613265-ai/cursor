# 审查意见答复

本分支用于存放**答复审查意见通知书**相关的 skill，与"专利撰写"（撰写申请文件）、"权利要求质检"（对已撰写权利要求书的质检）是并列的顶层分支，对应专利审查阶段的工作。

## 现有 skill

| skill | 路径 | 用途 |
|-------|------|------|
| 创造性答复意见陈述书撰写 | `创造性答复/意见陈述书撰写/` | 针对创造性（法22.3）**分步**撰写。逐条答复归入非显而易见性节（**短文照录、长文忠实概括**）；每项独权分别完成“区别技术特征 → **充分的非显而易见性** → **单独且深入的显著技术进步**”，两项独权不得用“同理”带过。**技术问题只在区别特征节完整论述一次**，后文不重复。技术进步按“特征→机制→效果→协同→相对进步”深入论述，有益效果统一写在对应独权的技术进步一节。区别特征须编号+权利要求原文；情形一将补充内容接到原独权末尾。先分析选法并用 plan 驱动逐步撰写；禁止一次出全文 |
