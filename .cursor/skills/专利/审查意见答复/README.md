# 审查意见答复

本分支用于存放**答复审查意见通知书**相关的 skill，与"专利撰写"（撰写申请文件）、"权利要求质检"（对已撰写权利要求书的质检）是并列的顶层分支，对应专利审查阶段的工作。

## 现有 skill

| skill | 路径 | 用途 |
|-------|------|------|
| 创造性答复意见陈述书撰写 | `创造性答复/意见陈述书撰写/` | 针对创造性（法22.3）**分步**撰写。**一通**以常规三段论（2.1→2.2→2.3）为骨架，审查员结论性观点嵌入 2.2 引述拆解；**二通及以后**以针对审查员核心观点的逐条作答为主战场，三段论节收束（2.1 仍完整）。手法须组合成**递进链条**而非堆砌；技术问题只在区别特征节完整论述一次，但非显而易见性必须以该问题为锚否定技术启示。有益效果归技术进步节。先判一通/二通、选法组合并用 plan 驱动；禁止一次出全文 |
