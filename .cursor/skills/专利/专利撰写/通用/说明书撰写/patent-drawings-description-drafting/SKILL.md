# 说明书"附图说明"部分撰写规范（所有案件类型通用）

## 核心规则：只能依据交底书已有的附图撰写

撰写"附图说明"部分时，**必须以交底书里实际存在的附图为基础**——撰写人通常不具备绘图能力，附图是由发明人提供的，不能凭空"发明"一张交底书里不存在的图。

具体要求：
1. 先核对交底书里实际有几张图、每张图分别画的是什么内容，附图说明部分严格按照这些已有的图来写，图的数量、顺序、每张图对应的内容都要跟交底书保持一致。
2. **不能凭空新增一张交底书里没有的图**——即使权利要求书后续经过修改、新增了交底书没有覆盖的技术特征（比如新增了某个从权对应的具体结构），也不能自己想象着编一张新图出来交差，充当"已有附图"来写。
3. 附图标记说明部分列出的标号，也要以交底书原图上实际标注的标号为准，不能虚构一个交底书里不存在的标号。如果权利要求里的术语后来改了名字（比如"内喷管"改成了"喷雾管"），可以沿用原图的标号数字，把配套的文字说明改成新术语，但不能改变标号本身指向的实际部件。

## 核心发明点缺少附图时的处理方式

如果权利要求书（尤其是独权和排在前面的关键从权）里的核心技术特征，在交底书现有的附图里**没有对应的图能够体现**，正确的处理方式是：

1. 先按交底书现有的附图，把"附图说明"部分能写的内容写完；
2. **写完之后，专门用一段话提醒撰写人：哪些核心发明点目前没有对应的附图，需要联系发明人补充绘制新的附图**——列清楚具体缺的是哪个技术特征、建议补充什么类型的图（结构示意图/局部放大图/展开图等），方便撰写人去跟发明人沟通要图。

**不允许**跳过这一步提醒，也不允许自己代替发明人"创作"一张凭空想象的新图混在附图说明里当成已有的图来写——这一点是本skill最容易被违反、也最需要注意的地方：即使能"想象"出一张合理的图应该长什么样，也不代表可以自己编出来当成交底书已有的附图。

## 标点符号和排版格式

"附图说明"部分的标点和排版要按以下模版：

- 每张图的描述之间用**分号"；"**连接，写成连续的一整段（每张图一行，行末用"；"，不要写成一句一段、句号收尾再空行分隔）；只有**最后一张图**的描述以**句号"。"**结尾。
- 附图标记说明另起一段，开头直接写"**附图标记说明：**"（用中文冒号，不要另起markdown标题层级），紧接着用"1、XX；2、XX；3、XX；……"的格式，全部标号写在同一段里，用"；"分隔，最后一个标号后用句号"。"收尾。

模版示例：

```
图1为本实用新型喷雾降尘装置的整体结构示意图；
图2为本实用新型的外罩与导喷套连接处截断局部结构示意图；
图3为本实用新型喷雾降尘装置的后视结构示意图；
图4为本实用新型的风罩与加固架连接处截断局部结构示意图。
附图标记说明：1、外罩；2、风叶；3、环管；4、外喷管；5、外雾化孔；6、喷雾管；7、喷孔；8、侧管；9、雾化喷头；10、导喷套；11、连接管；12、阀门；13、螺纹管；14、风罩；15、加固架；16、过滤网；17、电机。
```

## 方法类发明：第一张附图必须是方法流程图

如果本专利是方法类发明（权利要求以步骤S1、S2、S3……的形式撰写），**附图说明里的图1必须是方法流程图**，用于展示整个方法的步骤顺序和逻辑关系，之后再排列其他的结构示意图/局部放大图等。

如果交底书没有提供流程图（比如发明人只给了结构图，没给流程图），这属于"核心内容缺少附图"的情形，按上面的规则单独提醒撰写人去问发明人要一张流程图，不能自己画一个流程图逻辑出来充当"交底书已有的图"。

## 检查清单

- [ ] 附图说明里列出的每一张图，是否都能在交底书里找到对应的原图？有没有凭空编造的图？
- [ ] 图的数量、顺序，是否与交底书一致？
- [ ] 方法类发明，图1是否为方法流程图？如果交底书没提供，是否已经在提醒清单里列出？
- [ ] 权利要求书里的核心技术特征，有没有遗漏对应附图的情况？是否已经在撰写完毕后单独列出提醒？
- [ ] 附图标记说明部分列出的标号，是否都能在交底书原图里找到对应的标注？有没有虚构的标号？
- [ ] 标点和排版是否符合模版：图与图之间用"；"连接、只有最后一张图用"。"收尾；附图标记说明另起一段以"附图标记说明："开头，标号之间用"；"分隔、整段只在最后用一个句号？

## 应用范围

本规则适用于撰写或审核任意类型专利说明书"附图说明"部分。与 `patent-specific-embodiment-drafting` 配合使用：具体实施方式部分引用的附图标记，同样需要限定在交底书已有附图（或后续发明人补充提供的附图）范围内，不能引用一个不存在的图。
