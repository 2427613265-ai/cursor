# 说明书摘要撰写规范（所有案件类型通用）

## 核心规则

撰写说明书摘要时，**全文不得超过300字**（含标点符号）。超出300字的，必须删减至300字以内。

## 检查清单

- [ ] 说明书摘要全文（含标点）是否不超过300字？
- [ ] 摘要是否概括了独立权利要求的核心技术方案？
- [ ] 摘要是否点出了主要技术效果，且与独立权利要求实际解决的技术问题相对应？
- [ ] 专利名称、主题名称是否与权利要求书标题一致？

## 应用范围

本规则适用于撰写或审核任意类型（结构类、方法类、算法类）专利的说明书摘要。建议权利要求书定稿后再撰写摘要，写完后用本 skill 核对字数与内容对应关系。
