# 具体实施方式补充整体装配方式（所有案件类型通用）

## 适用场景

很多结构类案件，权利要求只保护某个**部件/组件本身**（例如"一种散热外壳""一种夹持机构""一种滤芯"），而**没有限定它如何与外部部件装配、连接、安装到整机或系统上**。但交底书、附图里通常会画出整机或安装关系，出现一批**安装类部件标记**（如转轴、转子、法兰、机架、螺栓、机臂、底座等）。

本 skill 解决的问题是：这种"权利要求没保护、但客观存在的整体装配方式"，要不要在具体实施方式里补充、以及怎么补充。

## 核心规则

### 要补充整体装配方式

即使权利要求没有限定装配关系，具体实施方式也**要补充**该部件如何装到整机/系统上，理由有三：

1. **充分公开**：本领域技术人员需要知道该部件如何安装、与哪些部件配合，才能完整实施本发明；只写孤零零一个部件、不交代它怎么装上去，公开不够充分。
2. **附图标记不成孤岛**：附图里画出的安装类标记（转轴、转子、法兰、机架、螺栓、机臂等），如果说明书通篇不提，就会变成 `说明书质检/说明书具体实施方式清楚性与逻辑一致性规范` 里的"孤岛特征"——附图上有、正文里找不到呼应。
3. **留支持基础**：将来若要把安装结构补入从属权利要求，或提分案申请，说明书必须原本就记载了这些装配关系，否则会因"超出原始申请文件记载范围"而不被允许。

### 分层写法：权1一句总述 + 文末可选实施例详述

补充整体装配方式时，按两个层次写，**避免把装配方式写成独立权利要求的必要技术特征**：

1. **第一层——权1（独权）实施例展开段末尾，用一句话总述**装配/连接关系即可。
   - 例如："所述外壳可通过连接法兰6与电机转子5固定连接，并随所述电机转子5同步旋转。"
   - **点到为止、不展开细节**，让读者知道这个部件大致怎么装上去、和谁配合，但不至于误以为某种特定装配方式是独权的必要技术特征。

2. **第二层——详细装配结构放到整个"具体实施方式"的最后**，用"进一步地，在另一实施例中"引出。
   - 详细展开安装盘、轴孔、紧固件、定位配合、与机架/机臂的连接、随转子同步旋转等具体结构与配合关系。
   - 这一层与 `未写入权利要求的技术方案在说明书中的编排` 中"独立方案放最后"的规则一致：整体装配方式属于权利要求主线之外的独立内容，集中放最后，不打断前面各权利要求实施例的主线。

### 不要做的两件事

- **不要把装配方式写进权1复述段**：复述段只原样复述权利要求的技术方案，而装配关系是权利要求没有的内容，只能放到展开段那一句总述、或文末可选实施例中。
- **不要在权2～权N每条从权实施例里重复写装配方式**：那样既冗余，又会冲淡各从权自身限定特征的主线。整体装配方式集中在"权1一句 + 文末一段"即可。

## 与相邻 skill 的分工

- 本 skill 负责"**要不要补、补什么内容（整体装配方式）、用什么分层结构补**"。
- `未写入权利要求的技术方案在说明书中的编排` 负责"未入权内容（含本 skill 的装配方式详述段）在全文里放哪个位置"——本 skill 的详述段落，正是按那条 skill"独立方案放最后"来落位的。
- 装配段落本身的清楚性（部件是否有呼应、有无孤岛、有无歧义），按 `说明书质检/说明书具体实施方式清楚性与逻辑一致性规范` 核对。

## 示范（承接无人机马达散热外壳案例）

权利要求只保护散热外壳本体（加强筋、环形外沿、导流齿、镂空通风槽等），未限定它如何装到电机上：

- **第一层（权1实施例展开段末一句总述）**：
  > 此外，如图1、图2所示，所述无人机马达散热外壳可通过连接法兰6与电机转子5固定连接，并安装于所述转子5的桨座侧，随所述电机转子5同步旋转，以使所述导流齿9与所述镂空通风槽7在马达运行时发挥导流散热作用。

- **第二层（文末可选实施例详述）**：
  > 进一步地，在另一实施例中，如图1、图2所示，所述散热外壳靠近中心的一侧还可设置中心安装盘，所述中心安装盘中心开设有与电机转轴2配合的轴孔……并通过紧固件与所述转子5固定连接，所述中心安装盘朝向所述转子5的一侧设置环形定位凸台以保证同轴度；所述散热外壳随所述电机转子5同步旋转……

## 检查清单

- [ ] 权利要求只保护部件/组件本身、未限定装配关系时，具体实施方式是否补充了该部件如何装到整机/系统上的整体装配方式？
- [ ] 附图中出现的安装类标记（转轴、转子、法兰、机架、螺栓、机臂等），是否都在正文中有呼应，没有成为孤岛？
- [ ] 装配方式是否采用了"权1实施例展开段一句总述 + 文末可选实施例详述"的分层写法？
- [ ] 有没有把装配方式错误地写进权1复述段？
- [ ] 有没有在权2～权N各从权实施例里重复写装配方式？
- [ ] 文末的装配详述段是否用"进一步地/在另一实施例中"等引导语标注为可选内容，且按"独立方案放最后"落位？

## 应用范围

适用于任意技术方案类型（结构类/方法类/算法类）说明书"具体实施方式"部分的撰写与质检，尤其常见于只保护单个部件/组件的结构类案件。与 `说明书具体实施方式撰写`（复述+展开）、`未写入权利要求的技术方案在说明书中的编排`（未入权内容的位置）配合使用。
