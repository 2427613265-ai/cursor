# 权利要求中自定义名词的就地定义规范

## 核心规则

权利要求中一旦出现**非公知的、撰写者自定义的名词**（自己创造的术语、被赋予特殊含义的普通词、本发明特有的参数或量），必须在**权利要求内部**用"其中，……"句把它的定义交代清楚，**不能把定义甩给说明书**。

**理由**：权利要求是界定保护范围的法律文本，应当能够**自足地被读懂**。如果一个自定义名词的含义必须翻到说明书才能理解，权利要求本身就不清楚、可读性差，也容易在侵权判定和无效程序中因"以说明书解释权利要求"产生争议。说明书当然也要有对应记载（用于支持），但权利要求这一层不能缺定义。

## 什么是"需要就地定义"的自定义名词

先区分两类词：

- **公知术语 / 客观物理量**（不需要就地定义）：本领域技术人员一看就懂的常规部件和公认物理量，如"螺栓""电机""处理器""长度""角度""温度""转速"。
- **非公知的自定义名词**（必须就地定义）：撰写者根据本发明自造或特殊约定的概念，读者无法仅凭常识准确理解，例如"调节比""稳定系数""响应等级""基准夹角""有效工作区"等。

**判断方法**：对权利要求中每一个名词/参数自问——"本领域技术人员不看本申请说明书，能否准确、唯一地理解这个词的含义？"如果不能，它就是需要用"其中"句就地定义的自定义名词。

## 就地定义的写法：用"其中，……"句

在首次引入该自定义名词的技术特征之后，紧接着用"其中，"引出定义分句，说明它是什么、怎么计算、由哪些已定义特征构成。

### 正确 / 错误示范

```
错误（定义只在说明书里，权利要求读不懂）：
  ……当所述调节比大于1.5时，锁定所述调节杆。
```

读者不查说明书，无法知道"调节比"是什么的比值，权利要求不自足。

```
正确（用"其中"句就地定义）：
  ……当所述调节比大于1.5时，锁定所述调节杆；
  其中，所述调节比为所述调节杆的伸出长度与所述壳体高度的比值。
```

也可以在名词首次出现处用括注给出简义，再在需要展开时用"其中"句补充：

```
  ……根据置信度对候选结果排序；
  其中，所述置信度为该候选结果与预设模板的匹配像素数占模板总像素数的比例。
```

## 与其他规范的关系

- 本 skill 是 `权利要求清楚性与逻辑一致性规范` 中"规则二：突然出现且不是客观清楚物理量的特征（人为定义的词）"的**落地写法**：那条规则负责**识别**哪些是需要解释的自定义名词，本 skill 规定**怎么写**——用"其中"句写在权利要求内部，而不是推给说明书。
- 若自定义名词是"预设值/参数/系数"，其**来源**（是给定的还是算得的、先获取后使用）由 `预设值与参数来源检查` 负责；本 skill 负责的是该名词的**含义定义**是否就地写清。两者角度不同，都要过一遍。
- 就地定义之后，说明书仍需对该名词有对应、更详细的记载以提供支持，二者不冲突；在说明书"具体实施方式"中，这段更详细的释义应排在对应实施例之前，见 `具体实施方式名词先释义后展开`。

## 检查清单

- [ ] 权利要求中每一个名词/参数，本领域技术人员不看说明书能否准确理解？
- [ ] 凡是非公知的自定义名词，是否在权利要求内部用"其中，……"句（或括注）给出了定义？
- [ ] 是否存在"定义只写在说明书、权利要求里直接使用"的自定义名词？——若有，应把定义补进权利要求。
- [ ] "其中"句的定义是否只使用了权利要求中已经出现/已定义的特征，没有再引入新的未定义名词（避免"用未定义解释未定义"）？
- [ ] 同一自定义名词在全文是否始终使用同一名称和同一定义，前后一致？

## 应用范围

适用于任意技术方案类型（结构类/方法类/算法类）权利要求的撰写与质检。撰写时应边写边检查每个自定义名词是否就地定义；质检时作为 `权利要求质检/权利要求书质检` 清楚性维度的专项检查项纳入使用。
