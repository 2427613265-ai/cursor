# 结构类·说明书撰写（预留分支）

本分支用于存放**结构类案件说明书撰写**相关的 skill。

结构类案件的说明书与权利要求书通常配套撰写（术语定义、部件名称需要在两份文档中保持一致），本分支与同级的 `权利要求撰写/` 是同一类案件下的两个文档维度，建议撰写时相互对照参考。

目前暂无内容，后续新增说明书撰写相关 skill 时，请在本目录下按照 `.cursor/skills/专利/专利撰写/结构类/说明书撰写/<中文名称>/SKILL.md` 的结构添加。

> 注意：不区分案件类型、所有类型通用的说明书撰写规范（比如"技术领域"部分的固定句式），请放到 `专利/专利撰写/通用/说明书撰写/` 下对应的子分支（`发明内容/`、`附图说明/`、`具体实施例/`），不要放在本目录（本目录只放结构类专属的规范）。
