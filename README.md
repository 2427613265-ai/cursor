# cursor

本仓库存放个人 Cursor Agent Skills，目前包含一套**中文专利撰写与质检 skill 体系**。

## 目录结构

skill 位于 `.cursor/skills/专利/` 下，克隆本仓库到用户主目录（`~`）即可让 Cursor 直接加载：

```
.cursor/skills/专利/
├── 专利撰写/
│   ├── 通用/
│   │   ├── 撰写策略/          # 专利申请文件撰写策略
│   │   ├── 权利要求撰写/       # 形式合规、清楚性、必要特征、主谓宾、指代、条件句、预设值来源、自定义名词就地定义
│   │   └── 说明书撰写/         # 技术领域/背景技术/技术问题/有益效果/具体实施方式/附图说明/名词先释义后展开
│   ├── 结构类/               # 结构类权利要求撰写、功能性限定使用边界
│   ├── 方法类/               # 预留
│   └── 算法类/               # 预留
├── 权利要求质检/              # 权利要求书质检（主控）、单条权利要求审查、引用结构检查
├── 说明书质检/                # 具体实施方式清楚性与逻辑一致性
└── 审查意见答复/              # 预留
```

## 使用方式

将本仓库克隆到 `~`（会生成 `~/.cursor/skills/专利`），Cursor 会自动发现这些 skill。
