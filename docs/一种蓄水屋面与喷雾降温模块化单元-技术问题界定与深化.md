# 一种蓄水屋面与喷雾降温模块化单元 —— 技术问题界定与深化

## 一、技术问题分层界定

### 表层需求（课题背景，过宽，不作为本专利技术问题）
- **P0**：夏热冬冷地区夏季屋面得热大、空调能耗高，需降温遮阳。

### 本方案（现独权核心特征）直接解决的问题

| 编号 | 技术问题 | 对应独权技术特征 |
|------|----------|------------------|
| P1 | 蓄水隔热与喷雾蒸发降温分属两套独立系统，既有屋面改造安装复杂、水源与蓄冷不共用 | 蓄水腔 + 喷雾组件，且喷雾取水自本单元蓄水腔 |
| P2 | 整仓/现浇蓄水屋面无法工厂预制、装配施工、局部检修更换 | 模块化单元 + 侧壁连通接口装配成片 |
| P3 | 夏季降温措施在冬季带来得热/结冰负担（课题核心矛盾） | 排空口，冬季排空停用 |

### 方案遗留、尚未针对性解决的更深问题（补强候选）
- **P4**：被动蓄水蒸发省水但降温弱、响应慢；主动喷雾强但耗水大——二者现仅"装在一起"，缺协同/按需切换手段。
- **P5**：既有屋面荷载受限→蓄水浅→蓄冷容量小→午后长时热峰蓄冷提前耗尽。
- **P6**：喷雾未蒸发水滴飘散流失，净水耗大、易污染邻近区域。
- **P7**：多单元拼装后边缘单元供水/水位不均，降温不匀。

## 二、主要技术问题（界定结论）

现独权核心 = 把蓄水与喷雾降温集成于同一"可装配、可连通、可排空"的预制模块，且喷雾自取本腔水。据此确定主要技术问题：

> **现有技术中蓄水隔热与喷雾蒸发降温分属两套彼此独立的系统，难以在既有建筑屋面改造中集成安装，且传统蓄水屋面不能工厂预制、装配拼装与局部检修更换，冬季亦需额外处理停用问题；如何提供一种将蓄水与喷雾降温集成于同一预制单元、能装配连通成片、并可排空停用的屋面降温模块。**

- 对应说明书"目的陈述"，与独权一一对应；P4–P7 不堆砌进目的陈述，留待从权/有益效果或补强后独权承载。

## 三、深化技术问题（补强方向，提升创造性）

现方案停在"工程集成/装配"层面，区别特征偏工程化、创造性有限。向下追问一层，本质是两对技术矛盾：

### 深化问题 A：节水 与 强化降温 的矛盾
- 被动蒸发省水弱、主动喷雾强费水；简单并置未解决"何时用、如何协同、如何不浪费水"。
- **深化表述**：如何在同一模块内使被动蓄水蒸发与主动喷雾协同，在保证峰值降温的同时最大限度减少净水耗与雾滴飘散流失。
- **引出的新特征方向（待确认）**：喷头朝向蓄水腔水面、壳体上沿设导流回落面 → 未蒸发雾滴回落腔内，形成"取水—雾化—回落"**半闭环循环**，把开环耗水喷雾变为半闭环，节水同时持续蒸发降温。

### 深化问题 B：低荷载浅蓄水 与 午后长时热峰散热需求 的矛盾
- 既有屋面荷载限制蓄水深度→蓄冷容量小→热峰未过蓄冷已耗尽；边缘单元供水不均。
- **深化表述**：如何在既有屋面荷载允许的浅蓄水深度下，提高有效蓄冷/散热能力并延长有效降温时段，且使拼装成片的多单元降温均衡。
- **引出的新特征方向（待确认）**：把现独权中仅用于"装配连通"的连通接口，升华为承担"跨单元水量与水温均衡"功能——多单元蓄水腔连通成一个**共享蓄冷水池**，局部热峰由整片水体削峰填谷、边缘单元由连通补水，兼解 P5 与 P7。

### 补强价值提示
深化 A+B 后，`连通接口`（原为装配特征）与 `喷头朝向/回落面`（原为从权细节）被赋予解决更深矛盾的功能，可由"工程拼装类实用新型"升级为"半闭环循环 + 共享蓄冷削峰"的系统性方案，区别技术特征更硬、创造性更强。
