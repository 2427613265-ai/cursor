## 一、汇报一句话定位

做一块**工厂预制、装配式快装**的屋面遮阳降温单元：单元内**蓄水**（水体热容缓冲 + 表面蒸发散热），单元上方**喷雾**（雾粒蒸发吸热 + 薄雾遮挡太阳辐射），单元之间**水路快接**拼成整片屋面；**冬季排空停用**，规避课题所述"夏季遮阳/冬季得热"矛盾。契合课题"夏热冬冷地区、既有建筑改造、装配式单元化、轻质材料"的产品路线。

## 二、建议锁定的主要技术问题

建议收束为与结构一一对应的 2～3 条（利于实用新型创造性表述）：

1. **既有建筑屋面改造**受结构荷载与防水限制，传统整仓/现浇蓄水屋面荷载大、防水风险高、无法局部更换、施工周期长；
2. 独立喷雾降温系统需另设水箱、管网、支架，与屋面构造脱节，安装复杂，蓄冷与水资源不共用；
3. 常规遮阳降温措施难以同时兼顾"夏季有效遮阳降温"与"冬季不增加得热负担"，且屋面遮阳产品缺乏标准化、装配化形态。

**拟锁定的主要技术问题（草案）**：
如何提供一种适于既有建筑屋面装配式改造、将蓄水隔热与喷雾蒸发降温集成于同一预制单元、单元间可水路连通并可排空、且冬季可停用的屋面遮阳降温模块。

## 三、核心技术方案（结构草案）

### 3.1 总体构成（独权级）

一种蓄水屋面与喷雾降温模块化单元，包括：

| 序号 | 部件 | 作用 |
|------|------|------|
| 1 | **模块壳体/框体** | 承重、限位、拼装定位；底部垫脚/支座，与既有屋面防水层隔离保护 |
| 2 | **蓄水腔** | 容纳设定深度冷却水，形成蓄热/蓄冷水体（隔热层） |
| 3 | **溢流口 / 水位限定结构** | 限定最高水位，超额雨水排出，防止超载（呼应既有屋面荷载限制） |
| 4 | **侧向快接管口**（进/出/连通） | 相邻单元串联或分区并联，连通成片蓄水层 |
| 5 | **喷雾供水支管 + 雾化喷头** | 取自本单元蓄水腔或公共干管，向单元上方喷雾 |
| 6 | **取水过滤件** | 防杂物堵塞喷头 |
| 7 | **排水/排空口** | 检修、**冬季排空停用**、换水 |

可选（从权/实施例，勿轻易并入独权）：微型泵、单向阀、浮球补水阀、温度/液位测点座、可开启网孔盖板、反光浮板、种植上层、太阳能供电等。

### 3.2 差异化结构点（创造性抓手，请发明人勾选）

**方案 A（主推）：单元内"蓄水腔 + 架空喷雾梁"一体化**
- 壳体中部为开口蓄水腔；
- 壳体上部两侧/对角设立柱支架，支承横跨单元的**喷雾梁**（内置供水管、下向或斜向雾化喷头）；
- 喷雾梁进水经过滤后连通蓄水腔（本单元自循环）或连通侧向供水干管（区段集中供水）；
- 四角设定位榫/卡扣，侧壁设快接接头（公母对插或活接）。
- 效果叙事：一块单元同时完成"蓄水隔热层"与"喷雾支架管网"安装，减少二次布管；雾粒先冷却上空与屋面辐射环境，回落水回流蓄水腔，减少外排浪费。

**方案 B：蓄水腔上覆可开启网孔遮阳/蒸发盖板**——盖板浅色/反光强化被动遮阳，喷头嵌盖板边缘形成"盖下雾层"，减少杂物落入。

**方案 C：双腔（喷雾清水腔 + 集雨溢流腔）**——过滤后补入清水腔，提升喷头可靠性。

**汇报建议**：主讲方案 A，B/C 作为替代实施例写入说明书（审查退路 + 防规避）。

### 3.3 拼装与水路逻辑（便于画图）

```
雨 / 补水
   ↓
┌──────────────┐   快接   ┌──────────────┐
│ 模块1 蓄水腔 │←──────→│ 模块2 蓄水腔 │  … 连通成片
│   ↑过滤取水  │         │              │
│ 喷雾梁/喷头  │         │ 喷雾梁/喷头  │
└──────────────┘         └──────────────┘
        ↓ 回流/溢流
     既有屋面排水系统
```

- **连通方式**：侧壁快接串联（整区水位一致）或总干管并联取水喷雾；
- **喷雾水源**：优先用单元蓄水，干旱时由补水阀补入；
- **回流**：未蒸发雾滴经壳体上沿导流坡回落蓄水腔。

### 3.4 工作过程（讲清"怎么降温、怎么避冬季矛盾"）

1. **夏季平时/夜间**：蓄水腔保持设定水位，水体热容压制屋面温度波峰，表面自然蒸发散热；
2. **夏季高温时段**：开启喷雾（手动阀或温控启停，控制逻辑不写入实用新型独权），雾粒蒸发吸热并对太阳辐射形成薄雾遮挡（契合"遮阳"课题）；
3. **降雨**：多余雨水经溢流口排出防超载，过滤后可留存供喷雾；
4. **冬季/检修**：打开排空口排尽蓄水，模块停用——**不增加冬季得热负担**，可单块吊离更换。

## 四、与课题"建筑遮阳"的衔接话术

| 课题关注点 | 本单元如何对应 |
|------------|----------------|
| 遮挡太阳辐射（遮阳） | 薄雾层 + 可选反光盖板/浅色壳体，削弱到达屋面的有效辐射 |
| 降低夏季空调能耗 | 蓄水隔断热流 + 蒸发冷却降低屋面外表面温度，减少室内得热 |
| **夏季遮阳/冬季得热矛盾**（合同核心难点） | **冬季排空停用**，仅在夏季工作，不产生冬季得热惩罚 |
| 既有建筑改造、城市更新 | 装配式单元、单块可换、荷载可控（溢流限位），适合既有屋面加装 |
| 轻质高强材料、装配式单元化 | 壳体可选轻质高强混凝土/金属/工程塑料，快接拼装（与合同产品路线一致） |
| 节水/绿色低碳 | 雨水利用、雾滴回落回收，优于大流量漫淋 |

## 六、建议的权利要求布局（策略预告，非正式权项）

**独权（产品）**：模块壳体 + 蓄水腔 + 溢流/水位限定结构 + 侧向连通接口 + 喷雾供水管及雾化喷头（喷头装于壳体上方支架/喷雾梁）+ 取水过滤 + 排空口。

**从权梯度（示例）**：
1. 喷雾梁架空高度 / 喷头朝向（下喷、斜喷、对喷）
2. 快接接头形式（密封圈活接、公母插接、带截止阀快接）
3. 导流回流结构（壳体上沿斜坡、回流孔）
4. 双腔（喷雾清水腔 + 集雨腔）
5. 可开启网孔盖板 / 反光浮板
6. 单元底脚与既有屋面隔离垫
7. 集成微型泵（或外置泵接口）
8. 液位/温度测点安装座（结构座，不含算法）

实用新型以**结构组合**为主；参数化/遗传算法等属课题子课题一内容，另行以论文/软著/发明专利承载，不塞入本实用新型。
