# 一种蓄水屋面与喷雾降温模块化单元 —— 技术方案构思（发明人汇报稿）

## 一、汇报一句话定位

做一块**工厂预制、现场拼装**的屋面降温单元：单元内部**蓄水**（显热缓冲 + 蒸发散热），单元上方/侧边**喷雾**（蒸发吸热 + 薄雾遮阳），单元之间**水路快接**，拼成整片屋面后形成「蓄水层降温 + 喷雾间歇强化降温」的复合遮阳降温系统。

面向场景建议（待确认）：平屋面、上人/不上人屋面、金属屋面局部改造、遮阳构架顶面、机房/地下室顶板等夏季热负荷突出部位。

---

## 二、建议解决的主要技术问题

向发明人汇报时，建议把「问题」收束到与结构方案一一对应的 2～3 条（实用新型创造性评价更清晰）：

1. **现浇/整仓蓄水屋面**：荷载大、防水风险高、检修难、无法局部更换，施工周期长。
2. **独立喷雾系统**：另设水箱、管网、支架，与屋面构造脱节，安装复杂，水资源与屋面蓄冷未共用。
3. **仅种植/仅蓄水模块**：静置蓄水降温有限，高温时段缺少主动蒸发强化；喷雾系统又难以模块化、标准化安装。

**拟锁定的主要技术问题（草案）**：  
如何提供一种便于工厂预制与现场拼装、能将蓄水隔热与喷雾蒸发降温集成在同一单元内、并实现单元间水路连通与可控排水的屋面降温模块。

## 三、核心技术方案（结构草案）

### 3.1 总体构成（独权级）

一种蓄水屋面与喷雾降温模块化单元，包括：

| 序号 | 部件 | 作用 |
|------|------|------|
| 1 | **模块壳体/框体** | 承重、限位、拼装定位；底部可设垫脚或支座，与屋面防水层隔离保护 |
| 2 | **蓄水腔** | 容纳设定深度的冷却水，形成蓄热/蓄冷水体 |
| 3 | **溢流口 / 水位限定结构** | 限定最高蓄水水位，超额雨水排出，防止超载 |
| 4 | **侧向快接管口**（进水/出水/连通） | 相邻单元水平串联或分区并联，形成连通蓄水层 |
| 5 | **喷雾供水支管 + 雾化喷头** | 取自本单元蓄水腔或公共干管，对单元上方空气/单元上表面喷雾 |
| 6 | **取水过滤件** | 防止杂物进入喷头堵塞 |
| 7 | **排水/排空口** | 检修、越冬排空、更换水质 |

可选（从权/实施例，勿轻易塞进独权）：水泵、单向阀、浮球阀补水、温度/液位传感器、遮阳盖板、反光浮板、透水盖板、种植槽上层、太阳能供电等。

### 3.2 推荐的「差异化结构点」（建议作为创造性抓手）

以下组合是相对现有「整仓蓄水屋面」「独立喷雾系统」「单纯蓄排水模块」更容易讲出区别的方向，请发明人勾选可行项：

**方案 A（主推）：单元内「蓄水腔 + 架空喷雾梁」一体化**

- 壳体中部为开口蓄水腔；
- 壳体上部两侧或对角设**立柱/支架**，支承横跨单元的**喷雾梁**（内置供水管、下向或斜向雾化喷头）；
- 喷雾梁进水经过滤后连通蓄水腔（本单元自循环）或连通侧向供水管（区段集中供水）；
- 单元四角设定位榫/卡扣，侧壁设快接接头（公母对插或活接）。

技术效果叙事：一块单元同时完成「蓄水隔热层」与「喷雾支架管网」的安装，减少二次布管；喷头离水面有固定高度，雾粒先冷却上空空气与屋面辐射环境，回落水可回流蓄水腔，减少外排水浪费。

**方案 B：蓄水腔上覆「可开启遮阳/蒸发盖板」**

- 盖板部分透空或网孔，减少杂物落入，同时允许蒸发；
- 盖板外表面浅色/反光，强化被动遮阳；
- 喷头可嵌在盖板边缘或盖板下方，形成「盖下雾层」。

**方案 C：双腔结构——清水腔 + 溢流回水腔**

- 内腔蓄清洁喷雾用水；外圈集雨、溢流、回水；
- 过滤后补入内腔，提升喷头可靠性。

**汇报建议**：主讲 **方案 A**，B/C 作为替代实施例写入说明书，便于审查阶段退路与防规避。

### 3.3 拼装与水路逻辑（便于画图）

```
雨 / 补水
   ↓
┌──────────────┐   快接   ┌──────────────┐
│ 模块1 蓄水腔 │←──────→│ 模块2 蓄水腔 │  … 连通成片
│   ↑过滤取水  │         │              │
│ 喷雾梁/喷头  │         │ 喷雾梁/喷头  │
└──────────────┘         └──────────────┘
        ↓ 回流/溢流
     屋面排水系统
```

- **连通方式**：侧壁快接串联 → 整区水位基本一致；或分区设总干管，各单元并联取水喷雾。
- **喷雾水源**：优先用单元蓄水；干旱时由总管补水阀补入。
- **回流**：喷雾未蒸发部分落回本单元或邻单元蓄水腔（壳体上沿设导流坡）。

### 3.4 工作过程（向发明人讲清「怎么降温」）

1. **平时/夜间**：蓄水腔保持设定水位，水体热容抑制屋面温度波峰，表面自然蒸发带走热量。  
2. **高温时段**：开启喷雾（可手动阀，或温控启停——控制逻辑可不写入实用新型独权），雾粒蒸发吸热，并对太阳辐射形成薄雾遮挡（遮阳课题的契合点）。  
3. **降雨**：多余雨水经溢流口排出，避免超载；过滤后可留存供下次喷雾。  
4. **检修/冬季**：打开排空口排尽蓄水，单元可单块吊离更换。

## 五、现有技术粗分（非正式检索，仅供汇报对照）

公开可见方向包括：整仓蓄水屋面、种植/蓄水模块隔热、采光屋面喷雾降温系统、陶瓷雾化屋面装置、雨水收集+喷淋系统等。  
**本方案拟强调的空白点**：把「蓄水腔、喷雾支架/管网、单元间快接连通、溢流排空」做成**同一标准化模块单元**，而不是「屋面整浇蓄水 + 另装喷雾管网」或「仅蓄排水盘」。

## 六、建议的权利要求布局（策略预告，非正式权利要求书）

**独权（产品）**：模块壳体 + 蓄水腔 + 溢流结构 + 侧向连通接口 + 喷雾供水管及雾化喷头（喷头安装于壳体上方支架/喷雾梁）+ 取水过滤。  

**从权梯度（示例）**：

1. 喷雾梁架空高度 / 喷头朝向（下喷、斜喷、对喷）  
2. 快接接头具体形式（密封圈活接、公母插接、带截止阀快接）  
3. 导流回流结构（壳体上沿斜面、回流孔）  
4. 双腔（喷雾清水腔 + 集雨腔）  
5. 可开启网孔盖板 / 反光浮板  
6. 单元底脚与屋面隔离垫  
7. 集成微型泵（或外置泵接口）  
8. 液位/温度测点安装座（结构座，不写算法）  

实用新型以**结构组合**为主；智能控制、算法建议另案或仅作说明书附属描述。
