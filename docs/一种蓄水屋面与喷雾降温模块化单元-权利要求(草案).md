# 一种蓄水屋面与喷雾降温模块化单元 —— 权利要求（草案）

> 说明：已采纳深化方案 A（半闭环回落节水），并按发明人意见进一步收宽独权——喷雾组件的水泵/供水管组成、排空口均移入说明书/从权，独权只保留壳体（向上敞口蓄水腔）、连通接口、支架、雾化喷头这一核心链。实体参数/结构细节均为撰写方构思，供发明人核对；正式申请前需完整检索。

---

## 独立权利要求（当前版）

**1.** 一种蓄水屋面与喷雾降温模块化单元，其特征在于，包括壳体、连通接口、支架和雾化喷头；

所述壳体的内部形成有向上敞口的蓄水腔，所述蓄水腔用于蓄存冷却水；

所述连通接口设于所述壳体的侧壁并与所述蓄水腔连通，所述连通接口用于与相邻的模块化单元的连通接口相对接，以使相邻两个所述蓄水腔相连通；

所述支架设于所述壳体并向上延伸至所述蓄水腔的上方；

所述雾化喷头安装于所述支架，位于所述蓄水腔的上方，且所述雾化喷头的喷雾方向朝向所述蓄水腔的敞口，以使所述雾化喷头喷出的、未被蒸发的雾化水回落至所述蓄水腔内。

---

## 独权设计说明

**独权保留的必要技术特征（4 个顶层部件）**

| 部件 | 保留理由 |
|------|----------|
| 壳体（内含向上敞口蓄水腔） | 蓄水隔热 + 敞口是雾滴回落的前提 |
| 连通接口 | 体现"模块化单元装配连通成片"，区别于整体现浇蓄水屋面 |
| 支架 | 发明人要求保留；支承喷头并使其位于蓄水腔上方 |
| 雾化喷头（朝向敞口喷雾、未蒸发水回落腔内） | 方案 A 半闭环核心区别点，兼实现喷雾蒸发降温 |

**本轮移入说明书 / 下放从权的内容（进一步扩大独权范围）**
- 喷雾组件的**水泵、供水管**及其连通关系（含"取水自本单元蓄水腔"）→ 移说明书，并以从权 4 固定作退路；
- **排空口**（冬季排空停用）→ 移说明书，并以从权 2 固定作退路；
- 溢流、过滤、喷雾梁、喷头角度、导流面、盖板、双腔、材料等具体结构 → 从权。

**由此带来的范围/创造性权衡（提示）**
- 独权不再限定供水方式（可为水泵、市政水压、位差供水等）与是否设排空口 → **保护范围更大**。
- 半闭环的"取水自本腔"移出独权后，独权层面的方案 A 主要由**"喷头朝向敞口喷雾 + 未蒸发水回落蓄水腔"**体现；此特征相对"独立喷雾系统朝外喷、不回收"仍是清晰区别，创造性可成立。若担心创造性偏弱，可将从权 4（取水自本腔）作为重点退路，答审时并入独权。

## 从属权利要求（建议布局）

**2.**（排空口·冬季停用退路）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，还包括排空口，所述排空口设于所述壳体的底部或侧壁的下部，并与所述蓄水腔连通。

**3.**（溢流限位·控荷载）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体的侧壁的上部设有溢流口，所述溢流口与所述蓄水腔连通，所述溢流口用于限定所述蓄水腔内冷却水的最高水位。

**4.**（喷雾供水·取水自本腔·半闭环退路）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，还包括水泵和供水管；所述水泵的进水端与所述蓄水腔连通；所述供水管的进水端与所述水泵的出水端连通，所述供水管的出水端与所述雾化喷头连通。

**5.**（取水过滤·防堵）根据权利要求4所述的蓄水屋面与喷雾降温模块化单元，其特征在于，还包括过滤件，所述过滤件设于所述水泵的进水端。

**6.**（架空喷雾梁·方案A细节）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述支架包括两个立柱和喷雾梁；两个所述立柱分别设于所述蓄水腔的相对两侧；所述喷雾梁的两端分别与两个所述立柱连接，且所述喷雾梁横跨于所述蓄水腔的上方；所述雾化喷头安装于所述喷雾梁。

**7.**（多喷头间隔布置）根据权利要求6所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述雾化喷头为多个，多个所述雾化喷头沿所述喷雾梁的长度方向间隔布置。

**8.**（喷雾角度·斜向下·强化回落）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述雾化喷头的喷雾方向与水平面之间形成向下的夹角。

**9.**（导流回落面·半闭环优选）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体的顶部边缘设有导流面，所述导流面朝向所述蓄水腔向下倾斜，用于将落在所述顶部边缘的雾化水导入所述蓄水腔。

**10.**（快接接头形式）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述连通接口包括相互配合的插接头与承插口，所述插接头与所述承插口之间设有密封圈。

**11.**（排水阀）根据权利要求2所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述排空口处设有排水阀。

**12.**（补水浮球阀）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体还设有补水口，所述补水口处设有浮球阀。

**13.**（可开启反光盖板）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述蓄水腔的上方盖设有盖板，所述盖板上开设有多个通孔，所述盖板的上表面为反光面。

**14.**（底脚隔离·护既有屋面防水层）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体的底部设有多个支脚。

**15.**（双腔）根据权利要求4所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述蓄水腔包括清水腔和集雨腔，所述清水腔与所述集雨腔之间设有过滤隔层，所述水泵的进水端与所述清水腔连通。

**16.**（材料·呼应课题）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体为轻质高强混凝土件或金属件。

## 对应的主要技术问题（补强后）

> 现有技术中蓄水隔热与喷雾蒸发降温分属两套独立系统、难以集成于既有屋面改造，且喷雾降温多为开环取用外部水源、未蒸发雾滴飘散流失、耗水量大；如何提供一种将蓄水与喷雾降温集成于同一可装配连通的预制模块，并使未蒸发的雾化水回落蓄水腔形成半闭环循环，从而在强化降温的同时降低净水耗。
