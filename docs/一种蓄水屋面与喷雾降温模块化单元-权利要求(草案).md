# 一种蓄水屋面与喷雾降温模块化单元 —— 权利要求（草案）

> 说明：实体参数、结构细节均为撰写方基于技术主线（方案A：蓄水腔 + 架空喷雾梁一体化）构思，供发明人核对修正；正式申请前需完整检索。

---

## 独立权利要求（草案）

**1.** 一种蓄水屋面与喷雾降温模块化单元，其特征在于，包括壳体、连通接口、排空口和喷雾组件；

所述壳体的内部形成有蓄水腔，所述蓄水腔用于蓄存冷却水；

所述连通接口设于所述壳体的侧壁并与所述蓄水腔连通，所述连通接口用于与相邻的模块化单元的连通接口相对接，以使相邻两个所述蓄水腔相连通；

所述排空口设于所述壳体的底部或侧壁的下部，并与所述蓄水腔连通；

所述喷雾组件包括支架、水泵、供水管和雾化喷头；

所述支架设于所述壳体并向上延伸至所述蓄水腔的上方；

所述水泵的进水端与所述蓄水腔连通；

所述雾化喷头安装于所述支架，且位于所述蓄水腔的上方；

所述供水管的进水端与所述水泵的出水端连通，所述供水管的出水端与所述雾化喷头连通。

---

## 独权必要技术特征核对（撰写依据）

对应技术问题：**如何将蓄水隔热与喷雾蒸发降温集成在同一可装配、可连通、可排空的屋面预制单元内。**

| 技术特征 | 是否必要 | 理由（去掉会怎样） |
|----------|----------|--------------------|
| 壳体 + 蓄水腔 | 必要 | 蓄水腔既是隔热蓄冷水体，又是喷雾水源；去掉则无蓄水降温、无水源 |
| 连通接口（侧壁、连通相邻蓄水腔） | 必要 | 体现"模块化单元装配成片、水路连通"，区别于整体现浇蓄水屋面；去掉则退化为孤立盒子 |
| 排空口 | 必要 | 支撑"可排空、冬季停用"（呼应课题夏遮阳/冬得热矛盾）；去掉则无法停用排空 |
| 喷雾组件（支架/水泵/供水管/雾化喷头） | 必要 | 实现喷雾蒸发降温并与本单元集成；水泵保证水自蓄水腔上送至喷头雾化，缺一不可 |
| 喷头位于蓄水腔上方、取水自本单元蓄水腔 | 必要 | 体现"喷雾与蓄水集成于同一单元"，区别于另设水箱的独立喷雾系统 |

**已刻意下放到从权、未写入独权的内容**（避免不必要缩小保护范围）：
- 溢流口 / 最高水位限定（荷载控制）
- 取水过滤件（防喷头堵塞）
- 支架的具体形式（立柱 + 架空喷雾梁）、喷头数量与朝向
- 顶部导流回流面、可开启网孔/反光盖板、双腔结构
- 快接接头具体形式、排水阀、浮球补水阀、底脚隔离垫、测点安装座、壳体材料

## 从属权利要求（建议布局，供进一步撰写）

> 以下为方向性草案，用于梯度防御与审查退路，可依发明人意见增删细化。

**2.**（溢流限位·控荷载）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体的侧壁的上部设有溢流口，所述溢流口与所述蓄水腔连通，所述溢流口用于限定所述蓄水腔内冷却水的最高水位。

**3.**（取水过滤·防堵）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述喷雾组件还包括过滤件，所述过滤件设于所述水泵的进水端。

**4.**（架空喷雾梁·方案A细节）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述支架包括两个立柱和喷雾梁；两个所述立柱分别设于所述蓄水腔的相对两侧；所述喷雾梁的两端分别与两个所述立柱连接，且所述喷雾梁横跨于所述蓄水腔的上方；所述雾化喷头安装于所述喷雾梁。

**5.**（多喷头·喷头朝向）根据权利要求4所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述雾化喷头为多个，多个所述雾化喷头沿所述喷雾梁的长度方向间隔布置，且各所述雾化喷头的喷雾方向朝向所述蓄水腔的上方水面。

**6.**（导流回落）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体的顶部边缘设有导流面，所述导流面朝向所述蓄水腔向下倾斜。

**7.**（快接接头形式）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述连通接口包括相互配合的插接头与承插口，所述插接头与所述承插口之间设有密封圈。

**8.**（排空阀）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述排空口处设有排水阀。

**9.**（补水浮球阀）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体还设有补水口，所述补水口处设有浮球阀。

**10.**（可开启反光盖板）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述蓄水腔的上方盖设有盖板，所述盖板上开设有多个通孔，所述盖板的上表面为反光面。

**11.**（底脚隔离·护既有屋面防水层）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体的底部设有多个支脚。

**12.**（双腔·方案C）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述蓄水腔包括清水腔和集雨腔，所述清水腔与所述集雨腔之间设有过滤隔层，所述水泵的进水端与所述清水腔连通。

**13.**（材料·呼应课题）根据权利要求1所述的蓄水屋面与喷雾降温模块化单元，其特征在于，所述壳体为轻质高强混凝土件或金属件。

## 待发明人确认/修正的点

1. 喷雾水源是否**限定取自本单元蓄水腔**（现独权如此写，强化"集成于同一单元"的区别点）；若实际采用区段集中供水干管，需调整独权。
2. 水泵是**每单元一泵**（潜水泵/管道泵）还是**区段共用泵**？若共用泵，需把"水泵"改为"供水接口"下放，独权改以"供水管进水端与蓄水腔连通"表述。
3. 连通接口是否确为侧壁**水平对接**；是否需要同时保留"总干管并联"实施方式（可作平行独权或从权）。
4. 从权 2～13 的取舍与补充（尤其溢流限位、过滤、材料是否上移）。
